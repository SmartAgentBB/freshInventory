# 냉프로 (NengPro) - 프로젝트 개요

## 프로젝트 제목
**냉프로(NengPro): AI 기반 스마트 식재료 관리 모바일 애플리케이션**

---

## 프로젝트 주제

### 개요
냉프로는 **AI 기반 식재료 인식 기술**과 **스마트 재고 관리 시스템**을 결합하여, 가정의 주방을 더욱 효율적이고 경제적으로 관리하는 모바일 애플리케이션입니다. Google Gemini AI를 활용하여 사진으로 식재료를 자동 인식하고, 신선도 추적, 소비 분석, AI 요리 추천 등의 기능을 제공합니다.

### 핵심 가치
- **식재료 낭비 감소**: 유통기한 알림과 소비 분석으로 낭비 최소화
- **요리 시간 단축**: 보유한 재료 기반 요리 추천으로 의사결정 시간 절감
- **스마트한 장보기**: AI 기반 자동 완료 처리와 필요 재료 추천
- **편리한 재고 관리**: 사진 한 장으로 여러 식재료 자동 인식 등록

### 기술 하이라이트
- **React Native + Expo**: iOS/Android 크로스플랫폼 개발
- **Google Gemini 2.5 Flash Lite**: 이미지 기반 AI 식재료 인식
- **Supabase**: 클라우드 데이터베이스 및 실시간 동기화
- **React Native Paper**: Material Design 3 기반 모던 UI
- **TypeScript**: 엄격한 타입 안정성으로 버그 최소화

### 주요 기능 (5가지)

#### 1️⃣ **스마트 재고 관리**
- AI 이미지 분석으로 여러 식재료 자동 인식
- 신선도 기반 D-day 색상 표시 (안전/주의/위험)
- 냉동/냉장 보관 방법별 자동 분류
- 남은양 슬라이더로 직관적인 수량 조절

#### 2️⃣ **스마트 장보기 리스트**
- AI 인식된 식재료 자동 완료 처리
- 실시간 배지로 미완료 항목 개수 표시
- 소비 완료 항목 재구매 추천
- 완료/미완료 항목 탭 분리 관리

#### 3️⃣ **AI 요리 추천**
- 보유 식재료 기반 레시피 자동 생성
- 단계별 조리법 제공
- 부족 재료 자동 장보기 추가
- 마음에 드는 레시피 북마크 저장

#### 4️⃣ **소비 분석**
- 사용/폐기 비율 시각화
- 월별 소비 패턴 히스토리 추적
- 낭비 최소화를 위한 데이터 기반 인사이트

#### 5️⃣ **보안 및 사용자 관리**
- 이메일 인증 회원가입
- Google/GitHub OAuth 소셜 로그인
- Supabase Row Level Security (RLS)로 데이터 격리
- 비밀번호 재설정 및 회원 탈퇴 기능
